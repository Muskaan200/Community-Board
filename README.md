# Web Development Project 1 - Travel Planning Resources

Submitted by: **Muskaan**

This web app is a responsive travel resource board that displays 10 curated tools and websites to help users plan their trips. Each resource is displayed as a card with an icon, description, and button linking to the site.

Time spent: **2** hours spent in total

---

## Required Features

The following **required** functionality is completed:

- [x] **The app has a cohesive, unique theme for events or resources relevant to a specific community**
  - [x] Header/title describing the theme is displayed
- [x] **At least 10 unique events or resources are displayed in a responsive card format**
  - [x] There are at least 10 cards displayed 
  - [x] The cards are displayed in an organized format (using Flexbox layout)
  - [x] Each card includes some information about the resource

---

The following **optional** features are implemented:

- [x] Buttons or links to a related resource are on each card component
  - [x] All cards have a button that opens the resource in a new tab
- [x] The site is responsive for both desktop and mobile formats
  - [x] Web app is tested in both desktop and mobile window sizes

---

The following **additional** features are implemented:

- [x] Icons were added to each card using `react-icons`
- [x] Custom button styling with hover effects
- [x] Clean and modern layout using Flexbox
- [x] Organized folder structure with reusable components
